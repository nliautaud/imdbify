imdbify

Convert a movie list to a IMDb "Export this list"-like CSV file.

Drop a file containing IMDb identifiers like tt0654974 to get a CSV file similar to the ones exported from IMDb, that you can use to feed services asking only for a IMDb export file (ex. icheckmovies, Trakt.tv trough TraktRater, SensCritique trough imdb2senscritique...).

Trakt.tv backup files from trakt-tv-backup will be further recognized to include user dates and ratings.

Data is retrieved trough OMDb API.
